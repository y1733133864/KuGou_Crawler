# 🎵 酷狗音乐播放器

一个基于 Python 开发的桌面音乐播放器，集成了酷狗音乐搜索和下载功能。

## ✨ 功能特性

- 🎵 **本地音乐播放** - 支持 MP3、WAV、OGG、FLAC 格式
- 🔍 **在线音乐搜索** - 通过酷狗 API 搜索歌曲
- ⬇️ **音乐下载** - 一键下载搜索结果中的歌曲
- 📋 **批量下载** - 支持批量下载所有搜索结果
- 📁 **播放列表管理** - 自动加载本地音乐文件夹
- 🎚️ **音量控制** - 可调节播放音量
- ⏯️ **播放控制** - 播放/暂停、上一首/下一首、停止
- 🖼️ **背景图片** - 支持自定义背景图片
